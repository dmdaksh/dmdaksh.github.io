# Daksh Maheshwari's Personal Website

This website is a work in progress. I will be adding more content and features in the future.

This is the repository for my personal website. It is built using [Jekyll](https://jekyllrb.com/), a static site generator. The website is hosted on [GitHub Pages](https://pages.github.com/).

This website is based on the [al-folio](https://github.com/alshedivat/al-folio) theme. For more information on how to use the theme, please refer to the [documentation](https://github.com/alshedivat/al-folio/blob/master/CUSTOMIZE.md).


## Running the website locally

To run the website locally, you need to have Docker installed on your machine. You can download Docker from [here](https://www.docker.com/get-started).

Once you have Docker installed, you can run the following command to start the website:

```bash
docker-compose up
```
